Build with blocks
Pick a tool on the right, then click a cell to apply it. Right-click removes (teleporter pair → thing → structure → pad paint → one step of terrain, in that order).

Drag to orbit · wheel to zoom · middle-drag / shift-drag / arrows / WASD to pan. In Select mode, drag an object to move it to another cell.

Walk drops you into the map in first person: gravity, working doors (Space/E), and live teleporters. Load an IWAD first for the full effect.

Cells tinted red have under 56 units of headroom — the player cannot fit there and will be stuck. Raise the ceiling or lower the terrain.

Heights are in Doom map units: the player is 56 tall (eye level 41) and can step up 24. So a 16 slab is a stair step, 32 is thigh-high (jumpable in modern ports, a ledge in vanilla), and each cell is 64×64 — about a meter and a half across. Terrain steps of 16 are walkable; 32 needs a Slab stair in between. Wall blocks reach the ceiling. Put a Door in a gap between walls. The Exit switch is a wall the player "uses" to end the level. Add a Player start before exporting.

Teleporters are objects (Things → Teleporters), numbered 1–4. Place two or more with the same number and each sends the player to its nearest same-number partner — chains of 3, 4, or more work too. Rotate one to set which way the player faces on arrival. Each number uses one of Doom's four pad textures, added automatically on export.

+ Level adds another map to the WAD — each level's exit switch leads to the next (MAP01 → MAP02 → …), and each level has its own size, ceiling, and sky.

Play the WAD: gzdoom -file episode.wad
